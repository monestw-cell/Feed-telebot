# بوت الأخبار التقنية المؤتمت بالكامل

بوت تيليقرام احترافي يعمل على منصة Render على مدار الساعة لفحص مصادر الأخبار التقنية، وأخبار الذكاء الاصطناعي، والهواتف (عربية وأجنبية) كل 15 دقيقة، وترجمتها وصياغتها ونشرها تلقائياً مع الصور والتصنيف بدون أي تكرار.

## الأدوات والتقنيات المستخدمة (100% مجاني وبدون APIs مدفوعة):
- **الكشط والاستخلاص:** Requests + BeautifulSoup4
- **الترجمة الآلية:** Deep Translator (Google Translate API المحلي)
- **قراءة الأخبار:** Feedparser (RSS Feeds)
- **منع التكرار الشامل:** قاعدة بيانات SQLite محلية مدعومة بفهرس للبحث السريع عبر الـ MD5 Hashes.
- **الجدولة ومنع التداخل:** APScheduler
- **خادم الحفاظ على النشاط:** خادم HTTP داخلي مدمج متوافق مع UptimeRobot.
