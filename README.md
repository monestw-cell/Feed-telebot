# Tech & AI News Telegram Bot

بوت تيليقرام شخصي يقوم بجلب الأخبار التقنية، وأخبار الذكاء الاصطناعي، والهواتف بشكل دوري ومختصر بدون تكرار وبدون استخدام أي APIs مدفوعة.

## الميزات:
- كشط واختصار محلي للمقالات باستخدام مكتبة `Scrapling`.
- نظام فحص دوري مجدول عبر الـ RSS للمواقع العالمية الكبرى.
- منع تكرار الأخبار باستخدام قاعدة بيانات `SQLite` محلية.
- يحتوي على خادم ويب داخلي مدمج متوافق مع منصة **Render** للبقاء يعمل 24/7 مع **UptimeRobot**.

## ملفات المشروع:
- `main.py`: الكود الأساسي للبوت، خادم الويب، والمجدول.
- `Dockerfile`: إعدادات الحاوية لرفع المشروع على Render.
- `requirements.txt`: المكتبات البرمجية المطلوبة.

## طريقة الإعداد على منصة Render:
1. ارفع هذه الملفات على مستودع (Repository) خاص بك في GitHub.
2. أنشئ حساباً على منصة **Render**.
3. اختر إنشاء **Web Service** جديدة واربطها بالمستودع الخاص بك.
4. في قسم الـ **Environment Variables**، أضف القيم التالية:
   - `TELEGRAM_TOKEN`: التوكن الخاص ببوتك من BotFather.
   - `CHAT_ID`: الأيدي الخاص بحسابك على تيليجرام لاستقبال الأخبار.
5. بعد نجاح الـ Deployment، انسخ الرابط العام الذي تمنحه لك منصة Render وضعه في موقع **UptimeRobot** لعمل Ping كل 5 دقائق لمنع الـ Free Tier من النوم.
